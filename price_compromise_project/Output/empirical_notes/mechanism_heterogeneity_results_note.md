# 机制与异质性检验结果备忘

本表将显著性检验的重点从工资项本身转向本文理论真正要求较强的环节：关税冲击是否通过投入产出网络推高价格，尤其是通过关键供应链连接推高中间品投入价格和总产出价格。

核心结果：

- 完整 IO 网络冲击对总产出价格的影响：系数 0.0642，p = 0.0237。
- Acemoglu 5% 强连接网络冲击对总产出价格的影响：系数 0.0637，p = 0.0209。
- 完整 IO 网络冲击对中间品投入价格的影响：系数 0.0724，p = 0.0044。
- Acemoglu 5% 强连接网络冲击对中间品投入价格的影响：系数 0.0714，p = 0.0061。

解释：这些结果说明，显著性最强的经验事实不是工资项单独下降，而是价格冲击沿关键投入产出连接传导，并首先体现在中间品投入价格与总产出价格上。本文可以据此论证：当低价贸易品和关键供应链价格缓冲被削弱时，劳动力再生产成本面对更高价格压力；工资端没有同步补偿，因此价格型阶级妥协的物质基础受到侵蚀。

写作建议：把价格和中间品投入价格结果作为主显著结果，把价格-工资缺口/实际工资下降作为方向一致的辅助证据。
